# ML-WEATHER PREDICTION DASHBOARD

This project is a Weather Monitoring Web Application that integrates Real-Time Weather API data with Historical Weather Data from a CSV dataset.
It provides users with live conditions such as temperature, humidity, wind speed, pressure, visibility, and UV index, along with visual analysis of how different weather factors correlate with each other.
